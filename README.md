# GENotification

Local Notification for iOS & macOS



## Requirements

- iOS 10.0+, MacOS 10.15+



## Installation

#### CocoaPods

```
pod 'GENotification'
```

#### SPM

```
.package(url: "https://github.com/Goeun1001/GENotification.git", from: "1.0.0")
```
